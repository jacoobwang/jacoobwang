Maven搭建javaweb项目

发表于 2019-08-14

背景

由于公司大部分项目都使用java做后端开发，曾经也撸过一段时间java，只是现在在做前端方面的开发！决定还是抽业余时间来写写Java！

几年前还是用 eclipse 来写java项目，没想到现在eclipse已经成为昨日黄花，而 Intellij Idea 才是新的王者！

创建项目

选择 Maven -> maven-archetype-webapp

然后下一步填写 GroupId 和 ArtifactId

GroupId: 一般分为多个段，这里我只说两段，第一段为域，第二段为公司名称。域又分为org、com、cn等等许多，其中org为非营利组织，com为商业组织。举个apache公司的tomcat项目例子：这个项目的groupId是org.apache，它的域是org（因为tomcat是非营利项目），公司名称是apache

ArtifactId: 表示你这个项目的名称

继续下一步，设置项目目录

接下来就是初始化项目，等待就好了….

初始化完成后会看到web项目的基本配置

配置Tomcat

完成上述配置并应用

然后点击右上角的绿色三角形箭头，等待tomcat启动,可以在下方看到tomcat的启动信息
