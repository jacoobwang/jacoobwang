高效使用chrome

发表于 2017-12-25

chrome一直以来都是我的最爱！相信很多开发人员都是用chrome来开发吧！谷歌一直被视为IT界最牛逼的公司之一！这里记录下我常使用的chrome插件😁

常用插件

JSON-handle

对JSON格式的内容进行浏览和编辑，以树形图样式展现JSON文档，并可实时编辑。对于我这种经常需要调试后端 rest 接口的来说简直太爽了。

下载地址：http://jsonhandle.sinaapp.com/

WEB 前端助手

支持各种前端操作，如代码美化、压缩、字符串编解码、生成二维码等等操作。

下载地址：https://www.baidufe.com/fehelper

Octotree

访问 github 必备插件，能够树状显示代码目录结构，简直棒棒的👍

下载地址：https://github.com/buunguyen/octotree

OctoLinker

当访问github时，它会将代码中的require，include或import转化成一个链接，点击链接可以跳转到相应的库。这对我们阅读代码的体验就非常好啦！

下载地址：https://octolinker.github.io/

Redux DevTools

当我们使用redux来管理我们的数据流的时候，redux devtools能够非常清晰的看到存储在redux中的数据state变化，非常有用！

下载地址：https://github.com/zalmoxisus/redux-devtools-extension

One Tab

标签管理工具，如果你是一个处理座的童鞋，将非常喜欢这个插件！它将所有标签页转换成一个列表，节省内存！

下载地址：https://www.one-tab.com/

Save to Pocket

阅读管理工具，假如你阅读一篇博文的时候看到一半，突然要去开会，你可以把这篇文章加入pocket，然后通过手机上的pocket应用程序在开会的时候进行阅读。总之，它可以将你喜欢的文章同步到你的手机，方便随时随地阅读。

下载地址：https://getpocket.com
