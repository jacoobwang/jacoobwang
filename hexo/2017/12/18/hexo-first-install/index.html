终于决定还是搞一个hexo静态博客

发表于 2017-12-18

好几年前我有一个用php撸的博客，用了一段时间后，转用wordpress。毕竟当时wordpress太火爆，一度靠wordpress还赚了一点点外快！时钟转到2017年，wordpress貌似不再流行，php也不再流行。在网络上瞎逛，看到很多人用hexo，遂决定用hexo来建一个博客！

放弃wordpress还有另外一个原因，php站点毕竟是需要服务器资源的，就算是最便宜的搬瓦工一年也要耗费几十块大洋来支付费用，屌丝啊，总是不想付费。而刚好看到github pages不用钱，心中略喜，finally决定就是hexo拉。
